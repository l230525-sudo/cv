# My CV Website

A responsive, professional CV/Resume website built with HTML5 and CSS3.

## Features

- **Responsive Design**: Looks great on mobile, tablet, and desktop devices.
- **Clean Layout**: Uses Flexbox for a modern, structured layout.
- **Professional Styling**: Consistent typography and color palette.
- **Micro-interactions**: Subtle hover effects on links and buttons.
- **Print Friendly**: Structured to be easily readable.

## Structure

- `index.html`: The main HTML document containing all the content.
- `css/style.css`: The stylesheet containing all the visual styles.

## Customization

1. **Personal Details**: Update the `<h1>`, `<h2>`, and contact information in the `<header>`.
2. **Profile Photo**: Replace the `src` attribute of the `<img>` tag in the `.profile-photo` div with your own image URL or local file path.
3. **Content**: Edit the text in the "About", "Education", "Skills", "Experience", and "Projects" sections to reflect your own background.
4. **Colors**: You can easily change the color scheme by modifying the CSS variables in the `:root` section of `css/style.css`.

## Usage

Simply open `index.html` in any modern web browser to view the CV.
